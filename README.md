# ReimbursementPilot AI

AI-Powered Travel Expense Automation

## Features

* Upload travel receipts
* Categorize receipts into Food, Hotel, and Travel
* Preview uploaded receipts
* Delete receipts
* Auto-convert receipts to PDF
* Generate organized ZIP package
* Trip summary dashboard
* Success notifications and confetti effects

## Tech Stack

* React
* Vite
* Tailwind CSS
* JSZip
* jsPDF
* FileSaver.js
* Canvas Confetti
